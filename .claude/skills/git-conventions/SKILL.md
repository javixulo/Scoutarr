# Git Conventions Skill

## Critical rule

**No commit or push without explicit project owner approval.** This rule is absolute. No agent may bypass it under any circumstances.

---

## Commit message format

Follows the [Conventional Commits](https://www.conventionalcommits.org/) specification:

```
{type}: {short description} ({task reference})
```
